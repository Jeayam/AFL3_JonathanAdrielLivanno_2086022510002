# Student Portfolio - Jonathan Adriel Livanno

Proyek portofolio berbasis Multi-Page yang dibuat menggunakan Bootstrap 5.

## Fitur
*   Multi-Page Navigation : TMemiliki total 6 page (Home, About, Skills, Project, Gallery, Contact).
*   Responsive Design : Menggunakan Bootstrap Grid System agar dapat dipakai melalui perangkat desktop maupun mobile.

##  Teknologi yang Digunakan
*   **HTML5**: Struktur konten web.
*   **CSS3 & Bootstrap 5**: Styling, layouting, dan komponen UI (Navbar, Card, Button).
*   **JavaScript**: Validasi form, interaksi dinamis, dan feedback pengguna.
